How it works

Semantic Color Numbers

Lower numbers (0-3) = Background colors

Middle numbers (4-8) = Interactive elements

Higher numbers (9-12) = Foreground/text colors

This meaning stays consistent across themes!

// Light Mode Example
--cyan-2-hsl: 186 77% 77%; /* Light tint for backgrounds */

// Dark Mode (FLIPPED!)
--cyan-2-hsl: 189 84% 23%; /* Dark shade for backgrounds */

How it works

New Semantic Color System

--color-0 or --color = Most saturated primary color

Positive (1-10) = Background tones (shades in light, tints in dark)

Negative (--1 to --10) = Foreground tones (tints in light, shades in dark)

This creates natural contrast across themes!

// Light Mode Example
--cyan-0-hsl: [most saturated]
--cyan-3-hsl: [shade - darker for backgrounds]
--cyan--5-hsl: [tint - lighter for foregrounds]

// Dark Mode (FLIPPED meanings!)
--cyan-0-hsl: [same - most saturated]
--cyan-3-hsl: [tint - lighter for backgrounds on dark]
--cyan--5-hsl: [shade - darker for foregrounds on dark]

Customizing Base Colors

Using --base-white and --base-black

You can customize the base colors to add warmth or coolness to your entire palette!

/* Default (neutral) */
--base-white: hsl(0 0% 100%);
--base-black: hsl(0 0% 0%);

/* Warm palette - add orange/yellow tint */
--base-white: hsl(40 50% 98%);
--base-black: hsl(25 30% 5%);

/* Cool palette - add blue tint */
--base-white: hsl(210 30% 98%);
--base-black: hsl(220 40% 5%);

/* The extended shades/tints (indices 5-10, --5 to --10) */
/* will use color-mix() with these base colors, */
/* preserving your chosen warmth/coolness throughout! */
